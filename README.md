
# A discord bot that holds elections and gives a role to the user who has won the election(based on voting)

## Command list:

```
Config:

- !set_announce #<channel> 
- !set_voting #<channel>
- !set_duration <number> <minutes/hours>(default/no identifier is seconds)
- !set_role <role>
```

```       
Election:
- !start
- !end
- !vote @<user>
```
